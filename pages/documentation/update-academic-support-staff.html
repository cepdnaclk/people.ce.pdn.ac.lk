--- layout: documentation permalink: /documentation/update-academic-support-staff/ question: How to update Academic Support Staff page ? index: 160 title: How to update Academic Support Staff page ? ---
  1. Login to GitHub

  2. Go to Academic Support Staff page

  3. At the bottom of the page, click {% include documentation_image.html url_image='/images/docs/edit_this_on_github_staff.PNG' text='"Edit this page on GitHub"' %}

  4. Click the {% include documentation_image.html url_image='/images/docs/click_edit_inside_github.png' text='"Edit"' %} icon

  5. Make your changes to the file
    6. Variable Description
    staff_name Name of the person
    text_below_name Text thats written below the name
    url_image Path to the image. It should be uploaded to "images/staff/academic-support-staff/name1_name2.jpg"
    contact_number Contact number of the person
    on_duty Is this person on duty ? "true" or "false"
    email Email address of the person

  6. Click the {% include documentation_image.html url_image='/images/docs/propose_changes.png' text='"Propose changes"' %} at the bottom of the page

  7. Click the {% include documentation_image.html url_image='/images/docs/create_pull_req.png' text='"Create pull request"' %} button

  8. Click the {% include documentation_image.html url_image='/images/docs/create_pull_req2.png' text='"Create pull request"' %} button again

  9. Now an admin/a maintainer will approve these changes within 48hrs.
