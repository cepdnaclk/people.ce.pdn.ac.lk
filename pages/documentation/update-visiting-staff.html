--- layout: documentation permalink: /documentation/update-visiting-staff/ question: How to update Visiting Staff page ? index: 160 title: How to update Visiting Staff page ? ---
  1. Login to GitHub

  2. Go to Visiting Research Fellows or Visiting Lecturers page

  3. At the bottom of the page, click on Edit Details

  4. Make your changes to the file, by editing (or adding) following template. 
    5. 
{
    "name": "",
    "email": "",
    "position": "Visiting Research Fellow",
    "designations": [
        "DESIGNATION_1",
        "DESIGNATION_2"
    ],
    "image": "/images/staff/fellows/NAME.jpg",
    "linkedin": "#",
    "profile": "#",
    "google_scholar": "#",
    "researchgate": "#",
    "remarks": "#"
}

  5. Click the {% include documentation_image.html url_image='/images/docs/propose_changes.png' text='"Propose changes"' %} at the bottom of the page

  6. Click the {% include documentation_image.html url_image='/images/docs/create_pull_req.png' text='"Create pull request"' %} button

  7. Click the {% include documentation_image.html url_image='/images/docs/create_pull_req2.png' text='"Create pull request"' %} button again

  8. To upload the profile image of the person, visit here and upload the image, with square size picture not more than 1200px in width. (A detailed guide on uploading images available in here)

  9. Now an admin/a maintainer will approve these changes within 48hrs.
