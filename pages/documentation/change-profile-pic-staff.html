--- layout: documentation permalink: /documentation/change-add-staff-profile-pic/ question: How to change/add a new Staff Profile image ? index: 130 title: How to change/add a new Staff Profile image ? ---
  1. Go to the GitHub repository of this site

  2. Click {% include documentation_image.html url_image='/images/docs/click_fork.png' text='"Fork"' %}

  3. Click {% include documentation_image.html url_image='/images/docs/click_images.PNG' text='"images"' %}

  4. Click {% include documentation_image.html url_image='/images/docs/click_staff.PNG' text='"staff"' %}

  5. Click {% include documentation_image.html url_image='/images/docs/click_academic_staff.PNG' text='"academic_staff"' %}

  6. Click on "Add file" then on {% include documentation_image.html url_image='/images/docs/upload_files.PNG' text='"Upload files"' %}

  7. Drag and drop your files to the {% include documentation_image.html url_image='/images/docs/drag_and_drop.PNG' text='given place'%}
    • Give the file a descriptive name. Eg E18576.jpg, IshanFernando.jpg etc.
    • Only upload square photos. If you don't have square photos, Use this site to crop your photo to 1:1 aspect ratio

  8. Click {% include documentation_image.html url_image='/images/docs/commit_changes.PNG' text='"Commit changes"' %}

  9. Click on {% include documentation_image.html url_image='/images/docs/click_on_commit_msg.PNG' text='"Add files via upload"' %}

  10. Copy the new file path by clicking on the copy {% include documentation_image.html url_image='/images/docs/copy_new_file_path.PNG' text='button' %}

  11. Now go to the staff page that you want to change the profile picture of.
    • Click {% include documentation_image.html url_image='/images/docs/pages.PNG' text='pages' %}
    • Click {% include documentation_image.html url_image='/images/docs/click_pages_staff.PNG' text='staff' %}
    • Click {% include documentation_image.html url_image='/images/docs/click_academic_staff_2.PNG' text='"academic_staff"' %}
    • Click on the profile that you want to change the image of

  12. Click on{% include documentation_image.html url_image='/images/docs/edit_file.PNG' text='edit' %}

  13. Around {% include documentation_image.html url_image='/images/docs/edit_url_image_staff.PNG' text='line' %} 13 of the text, change the path of the "url_image" to the new path that was copied in step 10
    • Example, 
      url_image: images/staff/academic_staff/damayanthi-herath.jpg

  14. Click {% include documentation_image.html url_image='/images/docs/open_pull_request.PNG' text='"Contribute"' %} then click "Open pull request"

  15. Click {% include documentation_image.html url_image='/images/docs/create_pull_req.png' text='"Create pull request"' %}

  16. For the {% include documentation_image.html url_image='/images/docs/pull_req_title.PNG' text='Title' %}, write "Update profile image - name1-name2" replace 18456 with your registration number

  17. Click {% include documentation_image.html url_image='/images/docs/create_pull_req2.png' text='"Create pull request"' %}

  18. Now an admin/a maintainer will approve these changes within 48hrs.
