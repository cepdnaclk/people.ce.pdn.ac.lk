--- layout: documentation permalink: /documentation/change-add-student-profile-pic/ question: How to change/add a new Student Profile image ? index: 120 title: How to change/add a new Student Profile image ? --- There are 2 methods of editing a Student Profile page.
  1. Update using the Google Form
    • You will have to copy all the other existing values to the Google Form if you use this method.
    • View Google Form.
    • Note: Do not update using Pull Requests if you fill the google form.
  2. Update using a GitHub Pull Request
    • Easier if you are familiar with Github Pull Requests.
    • Follow the steps below to update using Pull Request
Update using GitHub Pull Request
  1. Go to the GitHub repository of this site

  2. Click {% include documentation_image.html url_image='/images/docs/click_fork.png' text='"Fork"' %}

  3. Click {% include documentation_image.html url_image='/images/docs/click_images.PNG' text='"images"' %}

  4. Click{% include documentation_image.html url_image='/images/docs/click_students.PNG' text='"students"' %}

  5. Click on your batch number

  6. Click on {% include documentation_image.html url_image='/images/docs/upload_files.PNG' text='"Upload files"' %}

  7. Drag and drop your files to the {% include documentation_image.html url_image='/images/docs/drag_and_drop.PNG' text='given place'%}
    • Give the file a descriptive name. Eg E18576.jpg, IshanFernando.jpg etc.
    • Only upload square photos. If you don't have square photos, Use this site to crop your photo to a 1:1 aspect ratio

  8. Click {% include documentation_image.html url_image='/images/docs/commit_changes.PNG' text='"Commit changes"' %}

  9. Click on {% include documentation_image.html url_image='/images/docs/click_on_commit_msg.PNG' text='"Add files via upload"' %}

  10. Copy the new file path by clicking on the copy {% include documentation_image.html url_image='/images/docs/copy_new_file_path.PNG' text='button' %}

  11. Now go to the student staff page that you want to change the profile picture of.
    • Click {% include documentation_image.html url_image='/images/docs/pages.PNG' text='pages' %}
    • Click {% include documentation_image.html url_image='/images/docs/students.PNG' text='students' %}
    • Click your batch number
    • Click on your registration number file name. Eg. e18568.html

  12. Click on{% include documentation_image.html url_image='/images/docs/edit_file.PNG' text='edit' %}

  13. Add the end of that page, change the path of the "image_url" to the new path that was copied in step 10
    • Example, 
      image_url: "images/students/e18/e18098.jpg"

  14. Click {% include documentation_image.html url_image='/images/docs/open_pull_request.PNG' text='"Contribute"' %} then click "Open pull request"

  15. Click {% include documentation_image.html url_image='/images/docs/create_pull_req.png' text='"Create pull request"' %}

  16. For the {% include documentation_image.html url_image='/images/docs/pull_req_title.PNG' text='Title' %}, write "Update profile image - E18456" replace 18456 with your registration number

  17. Click {% include documentation_image.html url_image='/images/docs/create_pull_req2.png' text='"Create pull request"' %}

  18. Now an admin/a maintainer will approve these changes within 48hrs.
